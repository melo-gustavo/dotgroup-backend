## Init project

- OBS: Para conseguir rodar o projeto é necessário ter instalado algumas ferramentas:
  - VS Code ou IDE de sua escolha;
  - Node JS (22.22.0) ou versão mais recente
  - Docker

- Preencha todos os dados da .env (a ideia é que tenha um arquivo ".env.example" em um código verdadeiro com os nomes das variáveis que precisam ser preenchidas, mas no nosso caso o .env está indo na raíz do projeto para facilitar a configuração do projeto)

- Instale todas as dependências necessárias rodando o comando abaixo:
  - npm install

- Após instalar todas as dependências necessárias do projeto, rode o comendo para subir container docker com o banco de dados SQL PostgreSQL:
  - docker compose up -d
